# ETAPA 2
## Caminho de dados da Mic-1

Para conseguir executar o programa, é importante estar no diretório de `etapa2.c` e ter os seguintes arquivos:

1. `programa_etapa2_tarefa1.txt` com as entradas para a tarefa 1
2. `programa_etapa2_tarefa2.txt` com as entradas para a tarefa 2

Após isso rode no terminal:

```
$ gcc etapa2.c -o etapa2
$ ./etapa2
```
